# ImmersingPicker 功能规划

## 项目概述
ImmersingPicker 是一个基于 Avalonia 和 FluentAvalonia 的学生选择器应用程序，主要用于教师随机选择学生回答问题，确保选择的公平性。

- [ ] 班级导入/导出功能
- [ ] 班级复制功能

## 技术栈
- **前端框架**：Avalonia + FluentAvalonia
- **后端**：.NET 6.0+
- **数据存储**：本地文件存储
- **UI 设计**：Fluent Design 风格

- 提供公平、透明的学生选择机制
- 简化教师课堂管理工作

- 支持多平台使用
