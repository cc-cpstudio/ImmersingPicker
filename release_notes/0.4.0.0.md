# ImmersingPicker v0.4.0.0-beta 发布说明

## 欢迎您使用 ImmersingPicker！

## 🎉 版本信息
- **版本号**: 0.4.0.0-beta
- **发布日期**: 2026-06-16
- **目标框架**: .NET 10.0
- **平台**: Windows、Linux
- **基线版本**: 0.3.0-beta (50cb517)

---

## ✨ 新增功能

### 🖥️ 全新编辑窗口
- **完整的班级管理界面**: 支持班级下拉选择、新建/删除班级
- **卡片式学生管理**: 添加/编辑/删除学生卡片，直观易用

### 🔤 默认字体设置
- **默认字体更换为 HarmonyOS Sans SC**

---

## 🔧 变更与重构

### 设置页面重构
- **统一采用 SettingsPageShell 控件**: 所有设置页面使用统一的标题和滚动布局
- **修复 SettingsExpander 用法**: 将错误的 `SettingsExpanderItem` 修正为 `SettingsExpander`

### 导航系统优化
- **Frame 导航方式更正**: 从直接设置 Content 改为 `Frame.Navigate`，修复导航错误

---

## 🗑️ 移除功能

### 自动更新系统
- **完全移除更新系统**: 删除了 NetSparkleUpdater 依赖及相关组件
- 删除 VersionHelper、SparkleUpdaterService、UpdateSettingsPage
- 从 AppSettings 中剔除更新相关属性与事件
- 移除设置窗口及通知中的更新入口

---

## 📦 主要依赖
- Avalonia UI 11.3.14
- FluentAvaloniaUI 2.5.1
- ClassIsland.Shared.IPC 2.0.3
- EPPlus 8.5.0
- Serilog 4.3.1
