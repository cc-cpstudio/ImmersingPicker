## 🚀 快速开始

### 首次使用
1. 跟随欢迎向导完成初始化
2. 创建班级或从 Excel 导入
3. 配置座位布局
4. 开始点名

### Excel 导入
1. 准备 Excel 文件（支持 .xlsx 格式）
2. 在编辑页面点击导入按钮
3. 选择文件并配置字段映射
4. 确认导入数据

### ClassIsland 联动
1. 确保 ClassIsland 已安装并运行
2. 在设置中启用 ClassIsland 联动
3. 数据将自动同步
4. 在 ClassIsland 中使用点名功能
