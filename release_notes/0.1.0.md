# ImmersingPicker v0.1.0 Pre-release 发布说明

## 欢迎您使用 ImmersingPicker ！

## 🎉 版本信息
- **版本号**: 0.1.0 Pre-release
- **发布日期**: 2026-03-20
- **目标框架**: .NET 9.0
- **平台**: Windows

---

## ✨ 核心功能

### 🎯 学生点名
- **公平点名**: 基于概率的公平选择算法
- **普通点名**: 随机选择模式
- **座位网格**: 可视化座位布局
- **学生管理**: 添加、编辑、删除学生

### 📋 班级管理
- 多班级支持
- 自动保存数据
- 自定义座位布局

### 🎨 界面特性
- Fluent Design 现代化设计
- 主题切换（浅色/深色/系统）
- 主窗口 + 悬浮窗双窗口
- 首次启动向导

### ⚙️ 主要设置
- **基础设置**: 主题、开机自启
- **点名设置**: 默认算法配置
- **安全设置**: 密码保护、U 盘校验
- **悬浮窗**: 可配置位置的快速访问窗口

### 📊 历史记录
- 完整的点名记录
- 时间戳记录
- 记录查看和管理

---

## 📦 主要依赖
- Avalonia UI 11.3.11
- FluentAvaloniaUI 2.4.1
- ClassIsland.Shared.IPC 2.0.3
- EPPlus 8.5.0
- Serilog 4.3.1

---

## 🚀 快速开始

### 首次使用
1. 跟随欢迎向导完成初始化
2. 创建或导入班级
3. 配置座位布局
4. 开始点名

### 基本操作
- **添加学生**: 编辑页面添加学生
- **配置座位**: 设置座位网格行列
- **开始点名**: 点击主界面点名按钮
- **查看历史**: 历史页面查看记录
