# ImmersingPicker v0.3.0-beta 发布说明

## 欢迎您使用 ImmersingPicker！

## 🎉 版本信息
- **版本号**: 0.3.0-beta
- **发布日期**: 2026-04-24
- **目标框架**: .NET 10.0
- **平台**: Windows、Linux
- **基线版本**: 0.2.0-beta (eb01902)

## ✨ 新增功能

###    Linux 支持
- ImmersingPicker 现在加入了 Linux 运行文件！
- 若使用 Linux 运行该程序，请尽量使用 X11 桌面环境。

### 🎨 沉浸式抽选界面
- **全新沉浸式抽选窗口**: 提供专注的抽选体验
- **全屏置顶显示**: 确保抽选过程清晰可见
- **窗口管理系统**: 实现主窗口与沉浸式窗口的无缝切换
- **焦点事件处理**: 智能管理窗口激活/失活状态
- **悬浮窗口同步**: 支持悬浮窗口状态同步

### 🔄 自动更新功能（不可用）
- **后端更新服务**: 实现版本检测和下载管理
- **前端更新界面**: 友好的更新提示和安装界面
- **下载管理器**: 支持断点续传和进度显示
- **更新安装器**: 自动完成更新安装流程

### 📇 学生管理增强
- **卡片式学生编辑**: 全新的卡片式编辑界面，替代表格式编辑

### 📥 班级导入优化
- **座位编辑对话框**: 新增 SecRandomSeatEditDialog 组件
- **导入前验证**: 检查班级名称是否已存在
- **座位信息设置**: 支持为每个学生设置行列座位信息
- **座位验证功能**: 确保输入的行列数据为有效数字
- **批量编辑提示**: 完善的导入结果反馈机制

### ⚙️ 抽选模式设置
- **抽选模式配置项**: 支持自定义抽选模式
- **相关逻辑完善**: 完成抽选模式的核心功能实现

### 🖥️ 编辑器窗口
- **编辑器功能恢复**: 完整的编辑器窗口支持
- **导航菜单集成**: 主窗口编辑器菜单项
- **OpenEditor 方法**: 快速打开编辑器窗口

## 🔧 架构优化

### 核心架构升级
- **升级到 .NET 10.0**: 使用最新的 .NET 运行时
- **跨平台支持改进**: 增强 ClassIslandIPCService 的跨平台能力

### 日志系统优化
- **日志格式改进**: 更清晰、更易读的日志输出
- **日志级别调整**: 优化日志输出策略

## 🚀 升级指南

### 从 0.2.0 升级
1. 下载 0.3.0-beta 安装包
2. 运行安装程序自动升级
3. 数据文件会自动保留

### 从源代码编译
```bash
# 确保安装 .NET 10.0 SDK
git clone https://github.com/ImmersingEducation/ImmersingPicker.git
cd ImmersingPicker
dotnet restore
dotnet build -c Release
```
